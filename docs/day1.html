Since it looks like I’m about to do a lot of puzzles, I might as well share my thinking process here. The list of Calories looks something like this:

1000
2000
3000

4000

5000
6000

7000
8000
9000

10000

Each line with a number represents the number of Calories from a food item. Food from the same basket is grouped together and separated by a blank line. So there’s 5 baskets here. (I actually needed to handle more baskets than this, but here’s a smaller example to help you understand!)

To get the basket with the most Calories, I need to split up the data into baskets, calculate the total Calories in each basket, and store that in a list. Something like this:

[6000, 4000, 11000, 24000, 10000]

Then I find the maximum value of this list, and we’re done! Not too bad, right? (For now…)

So we did just that, and managed to solve the puzzle in less than 5 minutes. We went back downstairs to report to Captain Elf. (It felt all too familiar…)

“Got the answer?”

“It’s not a great idea if everyone asks that one elf to share their food. They’ll run out of food really quickly, and then we’ll have no idea what to do. So instead, how about this? Let’s find the top 3 elves carrying the most Calories, and see how many Calories they’ve carried in total. That way, we could have a larger reserve, right?”

“Yeah, shouldn’t be too hard to find out. Let’s go back, Kris.”

Well… I’m not going to pretend that I’m surprised.

I guess the second part shouldn’t have been that hard, but… I struggled with finding the sum of the 3 largest numbers in the Calories list. At first, I thought that I could get the largest number in the list, add that to the total, and then remove it from the list. If I do that 3 times, I’ll get what I want, right?

Well, it works, but… it’s a bit clumsy.

That’s when Kris thought of a really good idea. We just need to sort the list in descending order, then take the sum of the first 3 items in the list. It’s really easy to do in Python:

return sum(sorted(list, reversed=True)[:3])

It can be done in just one line. Thanks, Kris!
